# Python Copy Paste PDF Corrector

## Deskripsi
Program untuk membenarkan hasil Copy Paste dari PDF yang berantakan.
Contoh:
- File input.txt
    ```txt
    Program untuk membenarkan hasil 
    Copy Paste dari PDF yang berantakan.
    ```

- File hasil output -> output.txt
    ```txt
    Program untuk membenarkan hasil Copy Paste dari PDF yang berantakan.
    ```
